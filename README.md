# Анализ аномалий налогового риска

Исследование синтетического датасета компаний (2023–2025): поиск отклонений от отраслевой нормы, приоритизация находок и формирование рекомендаций. Подробное задание — в [TASK.md](TASK.md).

## Данные

- `test_dataset.csv` — 500 компаний × 3 года (1 500 строк), 20 колонок
- `test_dataset_smp.csv` — демо-выборка (10 строк) для smoke-теста

## Запуск

### R (основной пайплайн)

```bash
Rscript run.R --render
```

Результаты сохраняются в `output/`:
- `findings.csv` — ранжированный список компаний
- `groups.csv` — связанные группы по учредителю
- `fig1_priority_map.png`, `fig2_typology_freq.png`, `fig3_affiliation_graph.png` — визуализации (в т.ч. двудольный граф аффилированности: учредители — круги, компании — квадраты)
- `report.html` — аналитический отчёт

## Методология (кратко)

Анализ ведётся через призму **налогового риска**. Восемь типологий (четыре ведущие, четыре фоновые) проверяются робастной статистикой относительно отрасли (`okved_section`, медиана/MAD, порог |z| ≥ 3,5). Приоритизация — двумерная матрица **вероятность × материальность** (80-й перцентиль по каждой оси). Подробности — в `docs/adr/`.

## Результаты

Проанализировано **500 компаний** (1 500 наблюдений company-year).

### Распределение по корзинам приоритета

| Корзина | Компаний | Доля |
|---|---:|---:|
| Срочное внимание (*urgent*) | 14 | 2,8% |
| Фоновый мониторинг (*background*) | 81 | 16,2% |
| Экономический сигнал, не аномалия | 54 | 10,8% |
| Без сигналов | 351 | 70,2% |

### Срабатывание типологий

| Типология | Компаний | Tier |
|---|---:|---|

### Ключевые находки

**Доминирующий паттерн — транзитные фирмы:** 24 компании с аномально высокой выручкой на сотрудника при штате 1–3 человека. 12 из 14 срочных находок попадают в эту категорию.

**Топ-5 по риск-скору (корзина *urgent*):**

1. C0449 — ПАО «Инвест-Ко-449» (зерновые): транзит + молодая крупная
2. C0327 — ПАО «Союз-Пром-327» (грузоперевозки): транзит + молодая крупная
3. C0419 — ООО «Кристалл-Холдинг-419» (логистика): транзит + молодая крупная
4. C0008 — ООО «Омега-Маркет-008» (аудит): транзит, 1 сотрудник
5. C0487 — АО «Южный-Корп-487» (аудит): транзит, 2 сотрудника

**Дивидендный вывод (срочные):** C0069 (IT-консалтинг) и C0251 (зерновые) — дивиденды при нулевой или отрицательной чистой прибыли.

**Связанные группы:** обнаружено 4 кластера с общим учредителем. Наиболее значим — **FND-43730** (6 компаний, разброс эффективной ставки налога ~32 п.п.) — признак перераспределения прибыли внутри группы.

**Экономические сигналы:** 54 крупные стабильные компании без единого флага риска — легитимный бизнес, не подлежащий эскалации.

### Рекомендации (кратко)

1. Первоочередная камеральная проверка транзитных фирм (начиная с C0449, C0327, C0419).
2. Проверка дивидендной политики C0069 и C0251.
3. Разбор группы FND-43730 на предмет трансфертного ценообразования.
4. Фоновые 81 компания — плановый мониторинг без срочной эскалации.

Полный отчёт с визуализациями: [output/report.html](output/report.html).
